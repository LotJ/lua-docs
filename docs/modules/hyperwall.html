hyperwall:getMaxShipClass ()
get the maximum ship class allowed through the hyperwall. Uses the same class numbers as the images and setship commands: 0 = fighter, 1 = transport, etc. All hyperwalls default to 10 (satellites) as the largest permitted ships. This virtually means that every mobile ship is blocked.

Returns:

    integer the maximum ship class allowed through the hyperwall
hyperwall:getMinShipClass ()
get the minimum ship class allowed through the hyperwall. Uses the same class numbers as the images and setship commands: 0 = fighter, 1 = transport, etc. All hyperwalls default to 10 (satellites) as the smallest permitted ships. This virtually means that every mobile ship is blocked.

Returns:

    integer the minimum ship class allowed through the hyperwall
